This folder inclues:
download_new.R : R script for downloading TCGA data
TMM.R : TMM Normalization from the following -> Robinson, Mark D., and Alicia Oshlack. "A scaling normalization method for differential expression analysis of RNA-seq data." Genome biology 11.3 (2010): 1-9.
calculate_TMB.py : calculates tumor mutation burden (TMB) in TCGA dataset
network_propagate_immune_biomarkers.py : network propagation using immunotherapy drug targets and protein-protein interaction (PPI) network
