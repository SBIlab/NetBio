"data" folder includes the following files.
"c2.all.v7.2.symbols.gmt" : Curated pathway gene sets from Molecular Signatures Database (https://www.gsea-msigdb.org/gsea/msigdb/).
"Marker_summary.txt" : List of genes for (i) immunotherapy targets, (ii) gene-based immunotherapy markers and (iii) tumor microenvironment-based biomarkers.
